# SimpleTex 公式识别工具

<SimpleTexOcr />

## ✨ 功能特性

### 📸 多种上传方式

- **文件上传**：点击选择本地图片文件
- **拖拽上传**：直接将图片拖拽到上传区域
- **剪贴板粘贴**：从剪贴板粘贴截图或图片
- **拍照识别**：使用摄像头实时拍照识别（需要浏览器授权）

### 🔍 智能识别

- 基于 SimpleTex API 的智能公式识别
- 支持复杂数学公式、矩阵、方程组等
- 自动优化识别结果

### 📝 结果处理

- **LaTeX 代码**：直接生成标准的 LaTeX 代码
- **实时预览**：使用 MathJax 渲染公式预览
- **一键复制**：快速复制 LaTeX 代码到剪贴板
- **处理日志**：详细记录每一步操作状态

## 📖 使用指南

### 第一步：准备公式图片

1. 确保公式清晰可读
2. 背景尽量简洁，避免复杂背景
3. 复杂公式可以分段处理

### 第二步：上传图片

选择以下任意一种方式上传：

- 点击"选择图片文件"按钮
- 拖拽图片到上传区域
- 截图后点击"从剪贴板粘贴"
- 点击"拍照识别"使用摄像头

### 第三步：开始识别

1. 点击"开始识别"按钮
2. 等待识别完成（通常需要 2-5 秒）
3. 查看识别结果和预览

### 第四步：使用结果

1. 检查 LaTeX 代码是否正确
2. 点击"复制"按钮复制代码
3. 粘贴到你的论文、文档或 LaTeX 编辑器中

## 🔧 技术说明

### API 配置

本工具使用 SimpleTex API 进行公式识别：

- **API 端点**：`https://server.simpletex.cn/api/latex_ocr`
- **认证方式**：使用 App ID 和 Secret 进行签名验证
- **返回格式**：JSON 格式的 LaTeX 代码

### 浏览器支持

- ✅ Chrome 80+
- ✅ Firefox 75+
- ✅ Safari 13.1+
- ✅ Edge 80+

### 注意事项

1. **网络要求**：需要稳定的网络连接访问 API
2. **图片限制**：建议图片大小不超过 5MB
3. **格式支持**：PNG、JPG、BMP 格式
4. **隐私说明**：图片仅用于公式识别，不会存储

## ❓ 常见问题

### Q: 识别结果不准确怎么办？

A: 尝试以下方法：

- 调整图片对比度
- 裁剪不必要的背景
- 确保公式清晰无遮挡
- 分段识别复杂公式

### Q: 无法访问相机怎么办？

A: 检查以下设置：

1. 确保浏览器已授权摄像头访问
2. 检查摄像头是否正常工作
3. 使用 HTTPS 协议访问（浏览器要求）

### Q: 出现 CORS 错误怎么办？

A: 这是浏览器的安全限制：

- 使用 Chrome 的 CORS 插件（开发环境）
- 或配置代理服务器
- 或将 API 调用移至后端

### Q: 支持哪些数学符号？

A: 支持标准的 LaTeX 数学符号：

- 基础运算：`+ - × ÷ =`
- 希腊字母：`α β γ π`
- 特殊符号：`∑ ∫ ∂ ∇`
- 矩阵和方程组
